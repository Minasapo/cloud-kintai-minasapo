# はじめに

このサイトは、本アプリのドキュメントポータルです。

クラウド<GlossaryTerm description="スタッフの勤務時間、出勤・退勤、休憩などの勤務実績全体を指します。">勤怠</GlossaryTerm>は、出退勤打刻、勤務時間・休憩管理、<GlossaryTerm description="スタッフが自身の勤怠記録に対して訂正を依頼する申請です。">修正申請</GlossaryTerm>と承認フローを一つの流れで行える勤怠管理システムです。
スタッフは日々の打刻や日報を記録でき、管理者は申請状況や勤怠状況を一覧で確認して対応できます。
このガイドでは、スタッフ向け・管理者向けの利用手順をロール別に整理しつつ、それぞれの中でユースケース別と機能別の両方から辿れるように案内します。開発者向けには、実装・運用情報を別セクションでまとめています。

## このサイトの構成

- スタッフ向け: ロール別の入口に加えて、ユースケース別と機能別の 2 軸で案内
- 管理者向け: ロール別の入口に加えて、ユースケース別と機能別の 2 軸で案内

リポジトリ直下の `docs/` 配下には補助資料や運用メモも残っていますが、恒久的に参照する仕様は順次このサイトへ統合します。
