# 一般向けドキュメント

このセクションは、クラウド勤怠を利用する一般ユーザー向けです。

## クラウド勤怠について

クラウド勤怠は、スタッフの毎日の勤務記録と、管理者による確認・承認を一つの流れで行える勤怠管理システムです。日々の打刻だけでなく、申請や日報まで同じ環境で扱えるため、記録漏れの防止と業務連携の効率化に役立ちます。

主な機能は次のとおりです。

- 出退勤打刻: 出勤・退勤・休憩の打刻を行い、勤務の開始から終了までを記録できます。
- 勤務時間・休憩管理: 勤務時間と休憩時間を確認し、日ごとの実労働時間を把握できます。
- 修正申請と承認フロー: 打刻漏れや記録ミスがあった場合は修正申請を行い、管理者が内容を確認して承認できます。
- 日報機能: その日の業務内容や進捗を日報として記録し、共有できます。
- 管理者機能: スタッフの勤怠状況や申請状況を一覧で確認し、必要な対応をまとめて行えます。
