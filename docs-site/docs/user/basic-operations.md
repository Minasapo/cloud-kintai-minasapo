# 基本操作

このページでは、日常業務でよく使う操作を、最短で確認できるようにまとめます。

## 想定読者

- スタッフ
- 管理者（基本操作の確認用途）

## まず最初に行うこと

1. 勤務開始時に打刻する
1. 当日または翌日に勤怠を確認する
1. 必要があれば修正申請を確認・対応する

## 操作手順へのリンク

- [画面遷移マップ](./navigation-map.md)
- [出退勤を打刻する](./time-recording.md)
- [勤怠を確認する](./attendance-check.md)
- [申請を確認する](./request-check.md)

## ロールごとの使い分け

- スタッフ: 主に打刻と自分の勤怠確認、修正申請
- 管理者: 申請確認・承認、スタッフ勤怠の管理
