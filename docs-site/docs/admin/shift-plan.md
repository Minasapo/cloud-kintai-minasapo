# シフト計画を作成する

このページでは、年間のシフト計画を作成、更新するための基本運用を説明します。
## この画面でできること

- 対象年を切り替えて計画を編集する
- 月ごとの申請期間を設定する
- 日ごとの必要人数を入力する
- 自動保存と手動保存で計画を確定する

## 基本操作

1. 画面上部で対象年を選択する
1. 各月の申請開始日と申請終了日を設定する
1. 必要な日付の人数セルを編集する
1. 保存状態を確認して、必要に応じて全体を保存する

## 日次運用の流れ

1. 当月と翌月の不足日を先に確認する
1. 不足が出やすい曜日から必要人数を調整する
1. 祝日や繁忙日の人数を重点的に見直す
1. 画面下部の保存時刻を確認して作業を完了する

## 確認観点

- 申請期間の開始日と終了日に抜け漏れがないか
- 必要人数が 0 のまま残っていないか
- 月をまたいだ運用で偏りが出ていないか
- 保存中表示が消えて更新完了になっているか

## 注意点

- 申請期間を有効にする月は、開始日と終了日の両方を設定してください。
- 複数ユーザーが同時に更新すると内容が重なる場合があります。表示が想定と違うときは、画面を更新して最新状態を確認してください。
- この画面は管理者向けです。権限がないユーザーは利用できません。
