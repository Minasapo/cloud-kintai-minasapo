# シフトを管理する

このページでは、管理者が `/admin/shift` 画面でシフトを確認・調整するための基本運用を説明します。

## この画面でできること

- スタッフごとのシフト配置を確認する
- 必要なシフト調整を行う
- 変更後の反映状況を確認する

## 日次運用の流れ

1. 当日と直近のシフト状況を確認する
1. 欠員や偏りがある枠を優先して調整する
1. 調整後に全体バランスを再確認する
1. 翌日分の不足がないことを確認する

## 確認観点

- 重要時間帯に必要人数が確保されているか
- 特定スタッフに負荷が偏っていないか
- 変更内容が意図どおり反映されているか

## 注意点

- `/admin/shift` の表示内容は、システム設定の表示モードにより切り替わる場合があります。
- この画面は管理者向けです。権限がないユーザーは利用できません。
- 日別詳細やスタッフ別詳細を確認する場合は、関連画面へ遷移して確認してください。
