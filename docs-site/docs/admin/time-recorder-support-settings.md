# 打刻補助設定を管理する

このページでは、打刻画面や勤怠修正入力を補助する設定をまとめて説明します。設定項目の棚卸しではなく、「何のために変えるか」と「どこに影響するか」を確認するためのページです。

## 対象設定

ここで扱う設定は、勤務ルールそのものを変えるというより、利用者が日々の打刻や修正で迷いにくくなるように補助するものです。見た目の調整に見える項目でも、入力ミスの減り方や問い合わせ件数に直結するため、変更前後で実画面を確認する前提で扱います。

- 打刻画面アナウンス
- 外部リンク
- 打刻理由
- クイック入力
- 残業確認

## 設定場所

| 設定 | 主な導線 |
| --- | --- |
| 打刻画面アナウンス | `/admin/master/time_recorder_announcement` |
| 外部リンク | `/admin/master/feature_management/links` |
| 打刻理由 | `/admin/master/feature_management/reasons` |
| クイック入力 | `/admin/attendances` 右上の `設定` > `申請・入力` |
| 残業確認 | `/admin/attendances` 右上の `設定` > `申請・入力` |

## 打刻画面アナウンス

アナウンスは、短期的な周知や一時的な注意喚起を利用者へ伝えるための設定です。恒久ルールを長文で載せるより、今すぐ見てほしい内容だけを簡潔に表示する使い方に向いています。

- 打刻画面上部に案内文を表示する設定です
- 表示を有効にしても、本文が空なら表示されません
- 利用者が閉じたアナウンスは、設定 ID と本文の組み合わせ単位でローカルに既読扱いになります

### 変更後に確認すること

- 打刻画面でバナーが表示されること
- 改行や長文でも崩れないこと
- 案内文を閉じたあと、再表示条件が想定どおりか

## 外部リンク

外部リンクは、打刻画面から別サービスや補助資料へ誘導するための設定です。リンクを増やしすぎると本来の打刻操作を邪魔するため、利用頻度が高いものだけに絞るのが基本です。

- 打刻画面などから参照するリンク集を管理します
- 各リンクはラベル、URL、アイコン、有効 / 無効を持ちます
- URL 内の `{staffName}` はスタッフ名に置換されます

### 変更後に確認すること

- 利用者向け画面でリンクが表示されること
- アイコンとラベルが意図どおりか
- `{staffName}` を使う URL が正しく展開されること

## 打刻理由

打刻理由は、修正申請の内容を管理者が読み解きやすくするための入力補助です。候補が多すぎると選びにくく、少なすぎると自由入力に寄りすぎるため、現場でよく使う理由に寄せて整理します。

- 勤怠修正時に選択できる理由候補を管理します
- 有効にした理由だけが入力補助として使われます

### 変更後に確認すること

- 勤怠修正画面で理由候補が表示されること
- 無効にした候補が出なくなること
- 文言が利用者にとって分かりやすいこと

## クイック入力

クイック入力は、同じ時刻を繰り返し入力する場面の操作負荷を下げるための設定です。候補を増やしすぎると逆に迷いやすくなるため、定時に近い代表値を中心に残すと使いやすくなります。

- 勤怠編集画面でワンタップ入力に使う時刻候補を管理します
- 出勤候補と退勤候補を別々に持ちます
- 有効にした候補だけがクイック入力ボタンとして表示されます

### 変更後に確認すること

- 勤怠編集画面で候補ボタンが表示されること
- 候補時刻が意図どおりに入力されること
- 不要な候補を無効化したときに表示されなくなること

## 残業確認

残業確認は、入力制御というより利用者への注意喚起に近い設定です。実運用のルールと画面上の案内がずれていると誤解を生みやすいため、文言や表示条件を変える際は現場ルールと一緒に見直します。

- 残業申請がない場合などに確認メッセージを出すかどうかを切り替えます
- 打刻や修正入力の案内メッセージとして作用する設定です

### 変更後に確認すること

- 対象画面でメッセージの表示有無が変わること
- 現場の運用ルールと案内内容が矛盾しないこと

## よくある場面

どの設定も単独で完結せず、利用者の画面とセットで確認して初めて意味が確定します。下の例は「何を変えるか」よりも「変更後にどの画面を見に行くか」を揃えるための目安です。

### 利用者への連絡を一時的に出したい

1. 打刻画面アナウンスを更新する
1. 表示を有効にする
1. 打刻画面で見え方を確認する

### 修正申請の入力を迷いにくくしたい

1. 打刻理由を整理する
1. クイック入力候補を見直す
1. 勤怠編集画面で入力導線を確認する

### 打刻画面から外部サービスへ案内したい

1. 外部リンクを追加する
1. ラベル、URL、アイコンを確認する
1. 実際の利用画面でリンク遷移を確認する
