# 設定項目一覧を確認する

このページでは、管理者向け設定画面で編集、切替、追加、削除できる項目を画面別に一覧化しています。

- 読者: 管理者
- 対象: 管理画面の設定メニューにある全 18 画面
- 用途: どの画面で何を設定できるかを素早く確認する
- 補足: `エクスポート` はオーナー権限向け、`開発者` は内部向け設定です

## 基本

### 集計対象月

- パス: `/admin/master/job_term`
- 目的: 締め日と集計対象期間を登録、編集、削除する

| 区分 | 項目 | 内容 | 備考 |
| --- | --- | --- | --- |
| まとめて登録 | 開始月 | 登録を始める月を指定する | `YYYY/MM` 形式 |
| まとめて登録 | 締め日 | 各月の締め日を指定する | `1〜31` |
| まとめて登録 | 登録月数 | 何か月分まとめて登録するかを指定する | `1〜12` |
| まとめて登録 | 調整方向 | 非稼働日に当たる場合に前倒しするか後ろ倒しするかを選ぶ | `前倒し` / `後ろ倒し` |
| まとめて登録 | 土日を考慮 | 土日を非稼働日として扱うかを切り替える | チェックボックス |
| まとめて登録 | 休日カレンダーを考慮 | 法定休日を締め日調整に使うかを切り替える | チェックボックス |
| まとめて登録 | 会社休日カレンダーを考慮 | 会社休日を締め日調整に使うかを切り替える | チェックボックス |
| まとめて登録 | プレビュー | 登録予定の集計対象月、期間、調整理由を確認する | 既存月はスキップ表示 |
| 一覧 | 集計対象月 | 登録済みの対象月を確認する | `YYYY年M月` 表示 |
| 一覧 | 有効期間 | 対象月に対応する開始日と終了日を確認する | 一覧列 |
| 一覧 | 作成日 | 登録日を確認する | 一覧列 |
| 行操作 | 編集 | 既存の集計対象月を編集する | ダイアログを開く |
| 行操作 | 削除 | 既存の集計対象月を削除する | 確認ダイアログあり |
| 編集ダイアログ | 集計対象月 | 編集対象の月を選び直す | 候補月から選択 |
| 編集ダイアログ | 開始日 | 対象期間の開始日を変更する | 必須 |
| 編集ダイアログ | 終了日 | 対象期間の終了日を変更する | 必須 |
