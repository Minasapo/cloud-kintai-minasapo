# 基本操作

このページでは、日常業務でよく使う操作を、最短で確認できるようにまとめます。

## まず最初に行うこと

1. 勤務開始時に打刻する
1. 当日または翌日に勤怠を確認する
1. ワークフロー申請の提出状況を確認する
1. 必要があれば修正申請を確認・対応する
