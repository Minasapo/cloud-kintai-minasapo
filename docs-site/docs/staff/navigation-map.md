# 画面遷移マップ（スタッフ向け）

このページは、スタッフが目的ごとにどの画面へ進めばよいかをまとめた案内です。

## 最短導線

出勤前
→ 打刻画面
→ 出勤を打刻
→ 勤怠一覧で確認

打刻ミス
→ 勤怠一覧
→ 対象日を開く
→ 編集して申請

## 画面名ベースの導線表

| 目的               | 開始画面 | 次に開く画面 | 完了条件                                   |
| ------------------ | -------- | ------------ | ------------------------------------------ |
| 出勤を記録する     | 打刻画面 | 勤怠一覧     | 出勤時刻が一覧に反映されている             |
| 退勤を記録する     | 打刻画面 | 勤怠一覧     | 退勤時刻が一覧に反映されている             |
| 打刻ミスを修正する | 勤怠一覧 | 勤怠編集画面 | 申請ステータスが申請中または承認済みになる |
| 休憩漏れを修正する | 勤怠一覧 | 勤怠編集画面 | 休憩時間が正しく登録される                 |

## 出勤・退勤を記録したい

1. 打刻画面を開く
1. 現在の状態を確認して打刻する
1. 必要に応じて勤怠一覧で結果を確認する

参照: [出退勤を打刻する](./time-recording.md)

## 打刻内容を確認したい

1. 勤怠一覧を開く
1. 対象日を選ぶ
1. 開始時刻・終了時刻・休憩を確認する

参照: [勤怠を確認する](./attendance-check.md)

## 打刻ミスを直したい

1. 勤怠一覧から対象日を開く
1. 編集画面で時刻を修正する
1. 修正理由を記載して申請する

参照: [勤怠を修正する](./attendance-edit.md)

参照: [申請を確認する](./request-check.md)
