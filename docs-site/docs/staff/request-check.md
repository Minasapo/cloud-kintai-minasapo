# 申請を確認する

このページでは、勤怠修正申請の確認手順を説明します。

## 対象となる申請

- 打刻漏れの修正申請
- 時刻誤りの修正申請
- 休憩時間の修正申請

## 手順

1. 勤怠一覧または編集画面で申請状況を確認する
1. ステータスが申請中の場合は承認待ちとして扱う
1. 差し戻しや再申請が必要な場合は、内容を修正して再申請する

## 運用例

1. 申請後は翌営業日に状態を確認する
1. 差し戻し理由がある場合は理由を確認して再申請する
1. 承認後に勤怠一覧の時刻が更新されているか確認する
