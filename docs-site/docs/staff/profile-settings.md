# 個人設定を管理する

個人設定画面では、自分の通知設定、個人リンク、ログイン情報を管理できます。

## この画面でできること

- 一般設定で、自分の名前、権限、利用開始日を確認する
- 通知設定で、勤務開始メールと勤務終了メールの受信を切り替える
- 個人リンク設定で、ヘッダーから開く自分用リンクを登録する
- セキュリティで、パスワードを変更する

## タブごとの見方

### 一般設定

- 名前、権限、利用開始日を確認する
- 必要なときは、この画面からログアウトする

### 通知設定

- 勤務開始メールと勤務終了メールを切り替える
- 変更後は保存ボタンを押さずに自動保存される

### 個人リンク設定

- 表示名と URL を入力して、自分用のショートカットを登録する
- リンクの有効、無効を切り替える
- 不要になったリンクを削除する
- 表示名と URL がそろったリンクだけ保存される

### セキュリティ

- 現在のパスワードと新しいパスワードを入力して変更する
- パスワード変更だけは手動で `パスワードを変更` を押す

## 自動保存の見方

通知設定と個人リンク設定は、変更後約 1 秒で自動保存されます。画面内の表示で保存状態を確認してください。

- `保存待ち`: 変更を検知し、これから保存する状態
- `保存中...`: サーバーへ保存している状態
- `最終保存`: 最後に保存できた時刻

## 個人リンク設定の注意点

- 追加した行は、表示名と URL がそろうまで画面上に残る
- 入力途中の行は保存対象にならない
- URL は `https://` または `http://` から始まる形式で入力する

## 保存に失敗したとき

- エラー通知が出たら、入力内容と通信状態を確認する
- 内容を少し修正するか、再度入力して保存を再試行する
- パスワード変更で失敗した場合は、現在のパスワードと新しいパスワードの条件を確認する
