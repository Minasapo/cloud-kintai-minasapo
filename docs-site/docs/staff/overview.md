# スタッフ向けドキュメント

このセクションは、クラウド勤怠を利用するスタッフ向けです。

## クラウド勤怠について

クラウド勤怠は、日々の勤務記録を一つの流れで行える勤怠管理システムです。出退勤の打刻から申請まで同じ環境で扱えるため、記録漏れの防止と業務連携の効率化に役立ちます。

主な機能は次のとおりです。

- 出退勤打刻: 出勤・退勤・休憩の打刻を行い、勤務の開始から終了までを記録できます。
- 勤務時間・休憩管理: 勤務時間と休憩時間を確認し、日ごとの実労働時間を把握できます。
- 修正申請: 打刻漏れや記録ミスがあった場合は修正申請を行い、管理者の承認を受けられます。
- 日報機能: その日の業務内容や進捗を日報として記録し、共有できます。
